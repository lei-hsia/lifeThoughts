BitMask 基本操作
操作分为: 
1. 设置状态. 2. 清除状态. 3. 查询状态.
---
1. 设置状态

```
    xxxx xxxx 
OR  0000 0010
  = xxxx xx1x
status |= mask;
```

2. 清除状态
```
    xxxx xxxx
AND 1111 1101
  = xxxx xx0x
status &= ~mask
```

3. 查询状态
```
    xxxx xxxx
AND 0000 0010
  = 0000 00x0
(status&mask), 结果>0代表状态为on， ==0代表状态为off.
boolean isOn = (status & mask) > 0;
```

在看优秀代码时， 看到|=， &= ~， (status & mask) > 0的时候就可以立刻反映过来是这行代码的目的是
设置状态， 清除状态还是查询某个状态, 提高我们阅读代码的理解能力.
